# DSC-316 Machine Learning — Notebooks Collection

This repository contains a set of Jupyter notebooks used for learning and experimenting
with common machine learning algorithms and workflows (linear/logistic regression,
gradient descent, decision trees, SVMs, and a Titanic dataset example).

**Overview**

- **Purpose:** Educational examples and hands-on experiments for core ML concepts.
- **Audience:** Me

**Notebooks**

- [decision_tree.ipynb](decision_tree.ipynb): Decision tree algorithms, visualization and evaluation.
- [Gradient_Descent_1.ipynb](Gradient_Descent_1.ipynb): Implementation and visualization of gradient descent.
- [lin_reg_with_grad_descend.ipynb](lin_reg_with_grad_descend.ipynb): Linear regression solved via gradient descent.
- [lin_regression.ipynb](lin_regression.ipynb): Linear regression using scikit-learn and evaluation.
- [logistic_regression.ipynb](logistic_regression.ipynb): Logistic regression classifier with preprocessing.
- [logistic_regression_2.ipynb](logistic_regression_2.ipynb): Alternate logistic regression implementation/notes.
- [svm.ipynb](svm.ipynb): Support Vector Machine examples and kernel experiments.
- [titanic_dataset_practice.ipynb](titanic_dataset_practice.ipynb): End-to-end Titanic dataset EDA, feature engineering, and modeling.
- [practice1.ipynb](practice1.ipynb): Miscellaneous practice exercises and experiments.
